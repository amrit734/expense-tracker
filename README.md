Expense Tracker
By Amrit Stephen Anand

About
-------

The Expense Tracker App is a personal tracking tool that maintains records of the user's transactions. 
The project uses Javascript for dynamic updates, DOM manipulation and data storage.

Functionality
--------------

- Manage Transactions - Transactions may be added or removed
- Transaction History - Transaction history is recorded and displayed
- Balance Calculation - Total balance is displayed above the transaction History
- Data Persistence - Entered data is stored and can be viewed again after refreshing the page or exiting it
- Validation - Warning messages are displayed when required fields are left unfilled
- Sorting - Transaction records may be sorted based on various criteria
- Filter By Date - Transaction records may be filtered by date
- Category Selection - Categories may be assigned to transactions
- Summary - Summary of expenses can be displayed
- Export Data - Records may be downloaded in the form of a CSV file.

Instructions
-------------

- Enter details related to the transaction in the related fields
- Click the "Submit Query" button to add a transaction record and save data.
- Transactions can be removed from the list by clicking the "-" button next to the desired transaction
- Total Balance is displayed above the input form
- Use the "Sort By" dropdown to sort as required
- The filter form can be used to filter records by date if required
- Click the "Clear" button to clear data.
- Click the "View Summary" button to display income
- Click the "Create Transaction File" button to create a download link to produce a CSV file
